# HIP

## Identification

`__HIP__`

:   Identifies the language as HIP.

`__HIPCC__`

:   Identifies the language as HIP.

## Versioning

`HIP_VERSION_MAJOR`

:   An integer for the major version of HIP.

    Defined in `<hip/hip_runtime.h>`.

`HIP_VERSION_MINOR`

:   An integer for the minor version of HIP.

    Defined in `<hip/hip_runtime.h>`.

`HIP_VERSION_PATCH`

:   An integer for the patch version of HIP.

    Defined in `<hip/hip_runtime.h>`.
