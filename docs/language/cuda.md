# CUDA

## Identification

`__CUDA__`

:   Identifies the language as CUDA.

`__CUDACC__`

:   Identifies the language as CUDA.

## Versioning

`__CUDACC_VER_MAJOR__`

:   An integer for the major version.

`__CUDACC_VER_MINOR__`

:   An integer for the minor version.

`__CUDACC_VER_BUILD__`

:   An integer for the patch version.
