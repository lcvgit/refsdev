# OHOS

`__OHOS__`

:   Identifies the target operating system as OHOS.

`__OHOS_FAMILY__`

:   Identifies the target operating system as OHOS.

## Versioning

`__OHOS_Major__`

:   An integer for the major version of OHOS.

`__OHOS_Minor__`

:   An integer for the minor version of OHOS.

`__OHOS_Micro__`

:   An integer for the patch version of OHOS.
