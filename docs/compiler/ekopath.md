# EKOPath

## Identification

`__PATHCC__`

## Versioning

`__PATHCC__`

:   An integer for the major version of EKOPath.

`__PATHCC_MINOR__`

:   An integer for the minor version of EKOPath.

`__PATHCC_PATCHLEVEL__`

:   An integer for the patch version of EKOPath.
