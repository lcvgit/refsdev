# DJGPP

## Identification

`DJGPP`
`__DJGPP`
`__DJGPP__`

:   Identifies the compiler as DJGPP.

## Versioning

`DJGPP`
`__DJGPP`
`__DJGPP__`

:   An integer for the major version of DJGPP.

`DJGPP_MINOR`
`__DJGPP_MINOR`
`__DJGPP_MINOR__`

:   An integer for the minor version of DJGPP.
