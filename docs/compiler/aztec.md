# Aztec C Compiler

## Identification

`AZTEC_C`
`__AZTEC_C__`

:   Identifies the compiler as Aztec C.

## Versioning

`VERSION`
`__VERSION`

:   An integer for the version of Aztec C, encoded as a decimal integer with the general form of `<ABB>`, where:

    - `<A>` is the major version; and
    - `<BB>` is the minor version.
