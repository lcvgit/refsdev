# CompCert C

## Identification

`__COMPCERT__`

:   Identifies the compiler as CompCert C.

## Versioning

`__COMPCERT_VERSION__`

:   <**_ABB_**>~10~

`__COMPCERT_MAJOR__`

:   An integer for the major version of CompCert.

`__COMPCERT_MINOR__`

:   An integer for the minor version of CompCert.
