# PGI

## Identification

`__PGI`

:   Identifies the compiler as PGI.

## Versioning

`__PGIC__`

:   An integer for the major version of the PGI compiler.

`__PGIC_MINOR__`

:   An integer for the major version of the PGI compiler.

`__PGIC_PATCHLEVEL__`

:   An integer for the major version of the PGI compiler.
