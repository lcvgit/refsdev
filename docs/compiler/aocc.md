# AMD Optimizing C/C++ Compiler

<!-- Based on Clang and DragonEgg/Flang -->

## Identification

`__aocc__`

: Identifies the compiler as AOCC.

## Versioning

`__aocc_major__`

: An integer for the major version of AOCC.

`__aocc_minor__`

: An integer for the major version of AOCC.

`__aocc_patchlevel__`

: An integer for the major version of AOCC.
